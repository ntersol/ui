Form Generator

The form generator is a tool for creating dynamic and customizable forms with features such as:

  • Model-driven approach for easy form creation and management.
  • Valid JSON for easy use with a REST API
  • Responsive layout for optimal display on various devices
  • A variety of form fields to choose from including input, dropdown, radio, and more
  • Support for HTML and dynamic content, allowing for flexible and personalized forms
  • Dynamic visibility for showing and hiding fields based on user input
  • Data fields to display information from external sources such as a REST API
  • Customizable footer to add additional information or links.

Layout

Layout content types are used to build responsive form with containers, rows, and columns and involve dividing the form into separate sections (containers) and organizing the form elements into rows and columns within those sections.

There are 3 varieties of layout content types: containers, rows and columns. Containers are wrappers around other content types and are used to to group other elements or provide styling. Rows form the basis of responsive layouts and contain columns which hold content. Layout content types are not required, IE you can have form content directly in the form generator without either a container or rows.
