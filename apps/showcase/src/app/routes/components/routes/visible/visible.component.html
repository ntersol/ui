Visible

Adds or removes elements in the DOM based on responsive breakpoints. This method uses ngIf so that the elements are not actually present in the DOM when hidden as opposed to merely hiding them with CSS.

Setup

Import

Html

Typescript

Resize window to see effect. Text will appear in the boxes below at the appropriate breakpoints. Inspect element to see the components added/removed to the DOM

<div visible="sm">Content sm</div>

Content sm only

<div visible="md">Content md</div>

Content md only

<div visible="lg">Content lg</div>

Content lg only

<div visible="md-up">Content md-up</div>

Content md-up
