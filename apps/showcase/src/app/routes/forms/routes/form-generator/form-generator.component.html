Layout

Layout content types are used to build responsive form with containers, rows, and columns and involve dividing the form into separate sections (containers) and organizing the form elements into rows and columns within those sections.

There are 3 varieties of layout content types: containers, rows and columns. Containers are wrappers around other content types and are used to to group other elements or provide styling. Rows form the basis of responsive layouts and contain columns which hold content. Layout content types are not required, IE you can have form content directly in the form generator without either a container or rows.

Dynamic Content

By using Angular curly braces, this form generation tool enables you to dynamically insert user-entered data into static content areas of the form in realtime, providing a personalized experience for the user and making the form more engaging. Data comes from the form root. The properties should use Angular dot notation for accessing reactive form data, IE "loanApplication.borrowers.0.name.age".

Dynamic content is available in: HTML content and form field labels/prefixes/suffixes/hints.
