SmartDistinctUntilChanged

A custom RXJS operator that can be used to check to check if the current emission from an observable stream is the same as the previous emission. Unlike the standard "distinctUntilChanged" operator, this operator recursively evaluates arrays and objects to see if they match even if the memory value isn't identical via "===".

Features:

  • Works with RXJS observables to check for duplicates in an emission stream.
  • Recursively evaluates arrays and objects to see if they match even if the memory value isn't identical.
  • Supports primitive values, objects, and arrays.
  • Has a fallback solution for older browsers that don't support "structuredClone".
  • Has NodeJS/SSR support

Limitations:

  • May be costly for large objects or arrays, use distinctUntilChanged with a custom comparator instead
  • Will not work with classes or complex objects with methods or accessors on older browsers that do not support "structuredClone"
