Script Loader

A convenient way to load 3rd party script files as an observable.

Key Features:

  • Full support for RXJS goodness
  • Script will only be loaded once, subsequent calls will complete immediately without reloading the script
  • Supports concurrency so that only one script will be loaded even if there are multiple subscribers

Check the JS tab in the console to see the script loading
