Example of the shared inputs and properties available to all form fields

Inputs

Property Name Type Default Value Description
placeholder string | null null Standard html placeholder text
label string | null null Floating label that appears in front of the content and moves above it when focused
prefix string | null null An icon of text that will appear BEFORE the input
suffix string | null null An icon of text that will appear AFTER the input
hint string | null null Small text that appears beneath the control
automationId string | null null A unique ID to use to help facilitate automated testing. Can be different than ID if ID is fixed
id string | null '' Text to use for ID attribute
name string | null null Text to use for name attribute
styleClass string | null null Any css classes
style string | null null Any inline style
validators Forms.Validators | null null Validators to attach through the template. Not preferred.
disabled boolean | null Enable/disable this control, uses the form control method instead of template property. Setting the disabled property should be done in the formgroup itself but this allows a parent component to set it via an input which makes it easier to model drive
formControl FormControl new FormControl() Form control
control AbstractControl | null | undefined Using an @Input() of "formControl" has a lot of problems with strict templates. Attempting to use the following syntax fails the compiler test: [formControl]="form?.get('zipcode')" because the formControl directive does not support the return type of that get request (AbstractControl | null | undefined). This is a workaround for that limitation

Outputs

Property Name Type Default Value Description
onFocus EventEmitter<FocusEvent> New EventEmitter<FocusEvent>() When the input is focused
onBlur EventEmitter<FocusEvent> New EventEmitter<FocusEvent>() When the input is blurred
onClick EventEmitter<MouseEvent> New EventEmitter<MouseEvent>() When the input is clicked
onChange EventEmitter<t> New EventEmitter<t>() When data on the input is changed
onKeyup EventEmitter<KeyboardEvent> New EventEmitter<KeyboardEvent>() On keyup event on the input

Property Name Type Default Value Description
maxLength number | null null The maximum number of characters allowed by this input.
minLength number | null null The minimum number of characters allowed by this input.

Property Name Type Default Value Description
showButtons boolean false Whether to show or hide spinner buttons.
min number | null null The minimum number to allow (not minimum characters).
max number | null null The maximum number to allow (not maximum characters).
maxLength number | null null The maximum number of characters allowed by this input (not maximum number).
mode 'decimal' | 'currency' 'decimal' The mode of the input: 'decimal' to allow decimal points, 'currency' to prepend a currency symbol.
useGrouping boolean true Whether to use a comma to separate thousands/millions/etc.
minFractionDigits number 0 The minimum number of fraction digits to use.
maxFractionDigits number 2 The maximum number of fraction digits to use.
currencySymbol string | null null The currency symbol to use when mode is set to 'currency'.

Property Name Type Default Value Description
placeholder string 'mm/dd/yyyy' The standard HTML placeholder text to display in the input field.
inline boolean false Whether to display the calendar as inline. Default is false for popup mode.
showIcon boolean true Whether to show the calendar icon on the input field.
dateFormat string | null null Format of the date which can also be defined at locale settings.

Property Name Type Default Value Description
options Forms.FieldOptions[] | null null The options to display for the dropdown.
dataKey string | null null A unique ID used for trackFn.
optionLabel string 'label' The property name for the label.
optionValue string 'value' The property name for the value.
optionDisabled string 'disabled' The name of the disabled field of an option.
scrollHeight string '200px' The default height of the dropdown panel.
appendTo any null The DOM element to which to append the dropdown panel. Can be 'body' or another template variable.
virtualScroll boolean false Whether to use virtual scroll to handle a large list of options.
showClear boolean false Whether to show an "X" which lets the user clear the selected value.
insertEmptyOption boolean false If true, inserts an empty option with the label of the placeholder and a value of null at the top of the list. Essentially a selectable null option.
placeholder string '-- Please Select --' The placeholder text to display when no option is selected.
panelStyle string '' The style of the dropdown panel.
panelStyleClass string '' The CSS class to apply to the dropdown panel.

Property Name Type Default Value Description
unmask boolean true Specifies whether form controls with input masks should return the raw data or the masked/formatted data. If set to true, unmasking is enabled and the raw data will be returned. If set to false, the masked data will be returned.

Property Name Type Default Value Description
options Forms.FieldOptions[] | null null Specifies the options to be displayed in the dropdown.
dataKey string | null null Specifies the unique identifier for the options, used for the track function.
optionLabel string 'label' Specifies the property of the option to use as the label to be displayed.
optionValue string 'value' Specifies the property of the option to use as the value that will be bound to the form control.

Property Name Type Default Value Description
options Forms.FieldOptions[] | null null Specifies the options to be displayed in the dropdown.
multiple boolean | null null Specifies whether multiple options can be selected.
dataKey string | null null Specifies the unique identifier for the options, used for the track function.
fullWidth boolean true Specifies whether the dropdown should take up the full width of its container.
canUnselect boolean | null null Specifies whether an option can be unselected once it has been selected.

Property Name Type Default Value Description
rows number 4 The number of rows (vertical) displayed in the textarea.
maxlength number | null null The maximum number of characters allowed in the textarea. If null, there is no maximum limit.
columns number | null null The number of columns (horizontal) displayed in the textarea.
autoResize boolean false When present, textarea size changes as being typed

Property Name Type Default Value Description
dataType 'string' | 'number' | null string Specifies whether the input data should be returned to the model as a string or a number.
allowNineDigitCodes boolean | null false Specifies whether the input should allow for 9-digit zip codes instead of just 5-digit codes.
