File Uploader

This file uploader component allows users to easily manage file uploads. You will still need to perform the upload to your API after a user has selected their files. It features a intuitive and user-friendly design, with the following capabilities:

  • Support for multiple file selection
  • Drag and drop functionality for easy file selection
  • Real-time file validation, checking for file size and type restrictions
  • File previews, including image previews for image files
  • The ability to show a preview of the uploaded files and metadata such as file name and file size
  • The ability to show the corresponding icon for the file type using the file extension
  • Error handling for upload restrictions
  • The ability to remove a single file out of multiple selected
  • The ability to reload and display previously uploaded files
  • The ability to reorder files
  • The ability to add additional files after initial selection
