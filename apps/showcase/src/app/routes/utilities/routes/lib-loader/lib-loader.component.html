Lib Loader

A utility that eliminates all of the overhead of loading and using 3rd party javascript libraries. This utility will load the library as an observable and then return a properly typed instance of that library upon load.

Key Features

  • Returns library instance from an observable (uses the Script Loader utility)
  • Library will be properly typed
  • Library instance will be cached and will only be loaded on the first request
  • Supports optional and automatic loading of plugins
  • Loaded from a CDN (CDNJS via cloudflare)
  • Version is configurable
